# F.R.I.D.A.Y-AI-Personal-Assistant
# Overview
F.R.I.D.A.Y is an AI-powered personal voice assistant designed to run seamlessly in the background of your computer and help automate daily tasks using voice commands. Inspired by futuristic assistants, F.R.I.D.A.Y can understand commands, open applications, respond intelligently using AI, and assist with productivity tasks in real time.
# Built using
Python, Ollama 3.2, Speech Recognition, Text-to-Speech, Automation Modules
# Future Plans 
Face recognition support, Camera-based object detection, Smart home integration, Web dashboard, Custom wake word support, Multi-language voice assistant, Full system automation

# Installation

To set up F.R.I.D.A.Y on your computer, first download or clone this project from GitHub and open the project folder. Make sure Python is installed on your system. Then install all the required Python libraries using the requirements.txt file. After that, download and install Ollama, which is used to run the AI model locally on your computer. Once Ollama is installed, open the terminal and run the Llama 3.2 model. Finally, start the assistant by running the main Python file. After launching, F.R.I.D.A.Y will run quietly in the background and listen for your voice commands to perform tasks like opening applications, answering questions, and automating basic computer operations.
